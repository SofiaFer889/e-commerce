PRODUCTS

/api/auth/products (GET)

endpoint para obtener productos paginados

parametros:

  • limit (number): limite de elementos a obtener
  • page (number): numero de pagina
  • sort (String): tipo de orden
  • query (number): query personalizada de busqueda

/api/auth/:pid (GET)

endpoint para obtener un product por id

parametros:

  • pid (String) requerido Mongo ID valido

/api/auth/products (POST)

endpoint para crear un nuevo product

parametros:

  • title (String) requeridotitulo del producto
  • description(String) requerido descripcion del producto
  • code(String) requeridocodigo unico del producto
  • price(number) requerido precio del productp
  • stock(number) requerido stock del producto
  • category(String) requeridocategoria del producto
  • file(file) imagenes con extensiones validas (png, jpg, jpeg)

/api/auth/products/:pid (PUT)

endpoint para actualizar un product

parametros:

  • pid (String) requeridoid del producto valido de Mongo
  • title (String) requeridotitulo del producto
  • description(String) requerido descripcion del producto
  • code(String) requeridocodigo unico del producto
  • price(number) requerido precio del productp
  • stock(number) requerido stock del producto
  • category(String) requeridocategoria del producto
  • file(file) imagenes con extensiones validas (png, jpg, jpeg)

/api/auth/products/:pid (DELETE)

endpoint para eliminar un product

parametros:

  • pid (String) requeridoid del producto valido de Mongo
